# WOLF PSN LOOKUP

Terminal tool to lookup PlayStation profiles using psn-api.

## Install

Clone repo:

git clone https://github.com/TUUSUARIO/wolf-psn-lookup.git

cd wolf-psn-lookup

npm install

## Setup NPSSO

Edit lookup.js:

const NPSSO = "YOUR_NPSSO";

Get NPSSO from:
https://ca.account.sony.com/api/v1/ssocookie

## Run

npm start

Enter PSN ID when prompted.
